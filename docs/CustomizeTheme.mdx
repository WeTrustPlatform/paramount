# Customization

## Use ThemeProvider

You can customize the theme using the `Theme` object [See Themeinterface](https://github.com/WeTrustPlatform/paramount/blob/master/src/theme/ThemeInterface.ts) and passing it to `ThemeProvider`

```
import { ThemeProvider, createTheme } from 'paramount-ui'

const theme = createTheme({})

// in the outmost component insert the provider and pass Theme Object as value
<ThemeProvider value={theme}>
  ...
</ThemeProvider>

```

## Override Component Styles

Each component exposts `getStyles` callback, which can be used to override component styles.

```
import { Button } from 'paramount-ui'

const CustomButton = () => {
  return (
    <Button
      color="primary"
      title="Example"
      getStyles={(buttonProps, theme) => {
        const { color } = buttonProps;

        // Each Component may use multiple style objects. The API enables to override each node's style
        // VSCode intellisense would autofill available style fields; however, you can view the style spec of each component in their `Component*.styles.ts`
        return {
          buttonStyle: { ... },
          textStyle: { ... },
          focusColor: color
        }
      }}
    >
  )
}

```
