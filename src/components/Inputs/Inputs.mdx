# Inputs

Paramount provides a set of common input elements for modern applications

## TextInput

Composes of React Native's `TextInput` component (i.e. you can pass its props). [See more details](https://facebook.github.io/react-native/docs/textinput)

### Usage

<Playground>
  <TextInput
    name="field"
    value={undefined}
    placeholder="Add your own placeholder"
    size="large"
    isDisabled={false}
    isInvalid={false}
    keyboardType="email-address"
    onChangeText={text => console.log('field', text)}
    onSubmitEditing={e => console.log('Submit on enter')}
    getStyles={(props, theme) => ({
      inputStyle: {},
      placeholderTextColor: theme.colors.text.muted,
    })}
  />
</Playground>

## IconTextInput

Add custom icons on the left or right within the input box. Composes of `TextInput`

<Playground>
  <Box marginBottom={24}>
    <State initial={{ value: '' }}>
      {({ state, setState }) => (
        <IconTextInput
          leftIcon={
            <View>
              <Icon name="menu" size={24} />
            </View>
          }
          rightIcon={
            <View>
              <Icon name="menu" size={24} />
            </View>
          }
          onChangeText={text => setState({ value: text })}
          value={state.value}
        />
      )}
    </State>
  </Box>
</Playground>

## ClearableTextInput

Adds a Close icon when there is an input which can be cleared. On clearing, it will change text to `''` and `onClear` callback is called. Composes of `TextInput`

<Playground>
  <Box marginBottom={24}>
    <State initial={{ value: '' }}>
      {({ state, setState }) => (
        <ClearableTextInput
          onChangeText={text => setState({ value: text })}
          value={state.value}
          onClear={() => {
            console.log('Cleared!');
          }}
          placeholder="Clearable text input"
        />
      )}
    </State>
  </Box>
</Playground>

## CopyTextInput

Adds a Copy icon when there is an input, and upon clicking it, it will save the content to clipboard. Composes `IconTextInput`

<Playground>
  <Box marginBottom={24}>
    <State initial={{ value: '' }}>
      {({ state, setState }) => (
        <CopyTextInput
          onChangeText={text => setState({ value: text })}
          value={state.value}
          onCopy={text => {
            console.log('Copied', text);
          }}
          placeholder="Text input wiht copy"
        />
      )}
    </State>
  </Box>
</Playground>

## PhoneNumberInput

Appends a button to select country code

### Usage

<Playground>
  <State initial={{ countryCode: 'US', phoneNumber: '' }}>
    {({ state, setState }) => (
      <PhoneNumberInput
        header={
          <Spacing marginVertical={3} paddingHorizontal={2}>
            <Heading size="xxxlarge">Select your country</Heading>
          </Spacing>
        }
        onChangeCountryCode={countryCode => setState({ countryCode })}
        onChangePhoneNumber={phoneNumber => setState({ phoneNumber })}
        phoneNumber={state.phoneNumber}
        countryCode={state.countryCode}
        placeholder="Enter your phone number"
      />
    )}
  </State>
</Playground>
