# CopyTextInput

Adds a Copy icon when there is an input, and upon clicking it, it will save the content to clipboard. Composes `IconTextInput`

<Playground>

    <State initial={{ value: '' }}>

        <CopyTextInput
          onChangeText={text => setState({ value: text })}
          value={state.value}
          onCopy={text => console.log('Copied', text)}
          placeholder="Text input wiht copy"
          // ...TextInput props
        />
      )}
    </State>
  </Box>
