# Alert

Give feedback to the user about an action or state

## Usage

### Intents

<Playground>
  <Box paddingVertical={8}>
    <Alert intent="info" title="Info" description="Message" />
  </Box>
  <Box paddingVertical={8}>
    <Alert intent="success" title="Success" description="Message" />
  </Box>
  <Box paddingVertical={8}>
    <Alert intent="warning" title="Warning" description="Message" />
  </Box>
  <Box paddingVertical={8}>
    <Alert intent="danger" title="Warning" description="Message" />
  </Box>
</Playground>

### With action button

<Playground>
  <Box paddingVertical={8}>
    <Alert
      title="With action"
      description="Add any action to the right side of the alert"
      overrides={{
        Action: {
          component: () => (
            <Button title="Action" appearance="minimal" color="primary" />
          ),
        },
      }}
    />
  </Box>
</Playground>
