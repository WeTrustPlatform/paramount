### Customization

Using `getStyles` prop

```
ButtonStyles {
  touchableStyle: ViewStyle;
  textStyle: TextStyle;
  innerButtonWrapperStyle: ViewStyle;
  buttonContentWrapperStyle: ViewStyle;
}

getStyles={(ButtonProps, Theme) => ButtonStyles}
```

Markup

```tsx
<TouchableOpacity touchableStyle>
  <View innerButtonWrapperStyle>
    {iconBefore}
    <View buttonContentWrapperStyle>
      <Text textStyle />
    </View>
    {iconAfter}
  </View>
</TouchableOpacity>
```
