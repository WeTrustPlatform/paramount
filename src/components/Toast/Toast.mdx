# Toast

### Setup

Put `ToastProvider` at the top level component to wrap components that make use of `ToastContext`.

```tsx
<ToastProvider>{children}</ToastProvider>
```

### Usage

Paramount exposes a convenience hook `useToast` you can use to create toasts from anywhere in the application

```tsx
function MyComponent() {
  const { notify } = useToast();

  return (
    <Button
      onPress={() => notify({ title: 'Title', description: 'Description' })}
    />
  );
}
```

Alternatively, use `ToastContext` directly.

<Playground>
  <ToastProvider>

    </ToastContext.Consumer>
  </ToastProvider>
