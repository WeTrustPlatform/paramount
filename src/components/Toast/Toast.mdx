## Usage

ToastProvider should inserted at the top-level component so that it displays from within the window container. However, it is possible to display Toast within another component like in the below example.

You could additionally wrap the container component with `overflow="hidden"` so that it disappears correctly

Composes `Alert`

<Playground>
  <ToastProvider>
    <ToastContext.Consumer>
      {({ notify }) => (
        <>
          <Button
            onPress={() =>
              notify({
                title: 'Title',
                description: 'Description',
                isCloseable: true,
              })
            }
            title="Open default toast"
          />
          <Button
            onPress={() =>
              notify({
                description: 'Description',
                intent: 'success',
                title: 'Title',
              })
            }
            title="Open success toast"
          />
          <Button
            onPress={() =>
              notify({
                description: 'Description',
                intent: 'danger',
                title: 'Title',
              })
            }
            title="Open danger toast"
          />
          <Button
            onPress={() =>
              notify({
                description: 'Description',
                intent: 'warning',
                title: 'Title',
              })
            }
            title="Open warning toast"
          />
        </>
      )}
    </ToastContext.Consumer>
  </ToastProvider>
</Playground>
