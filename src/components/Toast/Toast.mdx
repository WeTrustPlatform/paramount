# Toast

## Setup

Put `ToastProvider` at the top level component to wrap components that make use of `ToastContext`.

```tsx
<ToastProvider>{children}</ToastProvider>
```

## Usage

Paramount exposes a convenience hook `useToast` you can use to create toasts from anywhere in the application.

<Playground>
  <ToastProvider>
    {/*
      We use `ToastContext.Consumer` here due to limitations in docs
      Use `const { notify } = useToast()` inside your function component
     */}
    <ToastContext.Consumer>
      {({ notify }) => (
        <Button
          onPress={() =>
            notify({
              title: 'Title',
              description: 'Description',
              duration: 100000,
            })
          }
          title="Open toast"
        />
      )}
    </ToastContext.Consumer>
  </ToastProvider>
</Playground>

## Customization

Use `getStyles` prop on **ToastProvider**

```tsx
ToastStyles {
  containerStyle: ViewStyle;
  wrapperStyle: ViewStyle;
}

getStyles={(ToastProviderProps, Theme) => ToastProviderStyles}
```

Markup

```tsx
<>
  {children}
  <View containerStyle>
    <View wrapperStyle>
      <Alert />
    </View>
  </View>
</>
```
