# Toast

## Setup

Make sure to use `ThemeProvider` at the top-level component.

```tsx
<ThemeProvider>{children}</ThemeProvider>
```

## Usage

Paramount exposes a convenience hook `useToast` you can use to create toasts from anywhere in the application.

<Playground>
  <ToastProvider>
    {/*
      We use `ToastContext.Consumer` here due to limitations in docs
      Use `const { notify } = useToast()` inside your function component
     */}
    <ToastContext.Consumer>
      {({ notify }) => (
        <Button
          onPress={() =>
            notify({
              title: 'Title',
              description: 'Description',
              duration: 5000,
            })
          }
          title="Open toast"
        />
      )}
    </ToastContext.Consumer>
  </ToastProvider>
</Playground>
