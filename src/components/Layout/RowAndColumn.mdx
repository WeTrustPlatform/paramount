## Row Props

<Props of={Row} />

## Row Customization

Using `getStyles` prop

```
RowStyles {
  rowStyle: ViewStyle;
}

getStyles={(RowProps, Theme) => RowStyles}
```

Markup

```tsx
<View rowStyle />
```

## Column Customization

Using `getStyles` prop

```
ColumnStyles {
  columnStyle: ViewStyle;
}

getStyles={(ColumnProps, Theme) => ColumnStyles}
```

Markup

```tsx
<View columnStyle />
```
