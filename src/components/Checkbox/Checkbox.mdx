# Checkbox

Checkbox for boolean values

## Usage

### Toggle

<Playground>
  <Toggle initial={false}>
    {({ on, toggle }) => (
      <Checkbox
        shape="circle"
        value={on}
        onValueChange={toggle}
        label="Label"
      />
    )}
  </Toggle>
</Playground>
