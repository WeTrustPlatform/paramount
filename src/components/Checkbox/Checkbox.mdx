# Checkbox

Checkbox for boolean values

### Usage

<Playground>
  <Toggle initial={false}>
    {({ on, toggle }) => (
      <FormField label="Checkbox" labelPosition="right">
        <Checkbox shape="circle" value={on} onPress={toggle} size="medium" />
      </FormField>
    )}
  </Toggle>
</Playground>

### Props

<Props of={Checkbox} />

### Customization

Using `getStyles` prop

```
CheckboxStyles {
  touchableStyle: ViewStyle;
  checkboxStyle: ViewStyle;
  checkColor: string;
}

getStyles={(CheckboxProps, Theme) => CheckboxStyles}
```

Markup

```tsx
<TouchableOpacity touchableStyle>
  <View checkboxStyle>{icon}</View>
</TouchableOpacity>
```
