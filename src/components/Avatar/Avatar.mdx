# Avatar

Display avatar image, with default for name initials

### Usage

<Playground>
  <Avatar
    source={{ uri: 'https://picsum.photos/100/100', width: 100, height: 100 }}
    // Defaults to name if source is not provided
    name="Bill Gates"
  />
</Playground>

### Props

<Props of={Avatar} />

### Customization

Using `getStyles` prop

```tsx
AvatarStylesProps {
  name?: string;
  color: AvatarColor;
  isSolid: boolean;
  size: number;
  sizeLimitOneCharacter: number;
}

AvatarStyles {
  containerStyle: ViewStyle;
  textStyle: TextStyle;
  imageStyle: ImageStyle;
}

getStyles={(AvatarStylesProps, Theme) => AvatarStyles}
```

Markup

```tsx
<View containerStyle>
  <Text textStyle>{initials}</Text>
  <Image imageStyle />
</View>
```
