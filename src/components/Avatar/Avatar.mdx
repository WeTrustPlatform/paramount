# Avatar

Display avatar image, with default for name initials

## Usage

<Playground>
  <Avatar
    source={{ uri: 'https://picsum.photos/200/200', width: 200, height: 200 }}
    // Fallbacks to name if source is not provided
    name="Bill Gates"
    size="large"
  />
</Playground>

## Props

<Props of={Avatar} />

## Customization

Using `getStyles` prop

```tsx
AvatarStyles {
  containerStyle: ViewStyle; // set height and width for custom avatar size
  textStyle: TextStyle; // set font size if you want it to match avatar size
  imageStyle: ImageStyle;
}

getStyles={(AvatarProps, Theme) => AvatarStyles}
```

Markup

```tsx
<View containerStyle>
  <Text textStyle>{initials}</Text>
  <Image imageStyle />
</View>
```
