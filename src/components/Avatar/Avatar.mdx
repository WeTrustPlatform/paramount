# Avatar

Display avatar image, with default for name initials

### Usage

<Playground>
  <Avatar
    source={{ uri: 'https://picsum.photos/200/200', width: 200, height: 200 }}
    // Defaults to name if source is not provided
    name="Bill Gates"
    size="large"
  />
</Playground>

### Props

<Props of={Avatar} />

### Customization

Using `getStyles` prop

```tsx
AvatarStyles {
  containerStyle: ViewStyle;
  textStyle: TextStyle;
  imageStyle: ImageStyle;
}

getStyles={(AvatarProps, Theme) => AvatarStyles}
```

Markup

```tsx
<View containerStyle>
  <Text textStyle>{initials}</Text>
  <Image imageStyle />
</View>
```
