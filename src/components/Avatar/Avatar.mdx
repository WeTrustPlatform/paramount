# Avatar

Display avatar image, with fallback to name initials

## Usage

### With image

<Playground>
  <Avatar
    source={{ uri: 'https://picsum.photos/200/200', width: 200, height: 200 }}
  />
</Playground>

### Using initials

<Playground>
  <Avatar name="My Initial" />
</Playground>

### Varying sizes

<Playground>
  <Box flexDirection="row">
    <Avatar name="Small" size="small" />
    <Avatar name="Medium" size="medium" />
    <Avatar name="Large" size="large" />
  </Box>
</Playground>
