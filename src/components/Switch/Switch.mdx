# Switch

### Usage

<Playground>
  <Box>
    <Toggle initial={true}>
      {({ on, toggle }) => <Switch value={on} onPress={toggle} />}
    </Toggle>
  </Box>
</Playground>

### Props

<Props of={Switch} />

### Customization

Using `getStyles` prop

```
SwitchStyles {
  touchableStyle: ViewStyle;
  containerStyle: ViewStyle & SwitchStylesRequired;
  circleStyle: ViewStyle & SwitchStylesRequired;
  backgroundColorOff: string;
  backgroundColorOn: string;
  circleColorOff: string;
  circleColorOn: string;
}

getStyles={(SwitchProps, Theme) => SwitchStyles}
```

Markup

```tsx
<TouchableOpacity touchableStyle>
  <View containerStyle>
    <View circleStyle>{icon}</View>
  </View>
</TouchableOpacity>
```
