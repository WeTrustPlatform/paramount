# Collapsible

Component to reveal and hide content

### Usage

<Playground>
  <Toggle initial={false}>
    {({ on, toggle }) => (
      <Collapsible
        title={on ? 'Close collapsible' : 'Open collapsible'}
        isOpen={on}
        onOpen={toggle}
        onClose={toggle}
      >
        <Box
          marginVertical={16}
          justifyContent="center"
          alignItems="center"
          height={300}
          backgroundColor="#67c6bb"
        />
      </Collapsible>
    )}
  </Toggle>
</Playground>

### Customization

Using `getStyles` prop

```
CollapsibleStylesProps {
  isOpen: boolean;
}

CollapsibleStyles {
  touchableStyle: ViewStyle;
  outerWrapperStyle: ViewStyle;
  contentWrapperStyle: ViewStyle;
  textStyle: ViewStyle;
  iconWrapperStyle: ViewStyle;
}

getStyles={(CollapsibleStylesProps, Theme) => CollapsibleStyles}
```
