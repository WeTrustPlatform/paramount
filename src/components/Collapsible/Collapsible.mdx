# Collapsible

Component to reveal and hide content

## Usage

### Uncontrolled

<Playground>
  <Collapsible title="Toggle" initialIsOpen={false}>
    <Box
      marginVertical={16}
      justifyContent="center"
      alignItems="center"
      height={300}
      backgroundColor="#67c6bb"
    />
  </Collapsible>
</Playground>

### Controlled

<Playground>
  <Toggle initial={false}>
    {({ on, toggle }) => (
      <Collapsible
        title={on ? 'Close collapsible' : 'Open collapsible'}
        isOpen={on}
        onOpen={toggle}
        onClose={toggle}
      >
        <Box
          marginVertical={16}
          justifyContent="center"
          alignItems="center"
          height={300}
          backgroundColor="#67c6bb"
        />
      </Collapsible>
    )}
  </Toggle>
</Playground>
