# Slider

## Usage

<Playground>
  <Slider
    size="small"
    value={0.3}
    onValueChange={value => console.log('onValueChange', value)}
    onSlidingComplete={value => console.log('onSlidingComplete', value)}
    onSlidingStart={value => console.log('onSlidingStart', value)}
  />
</Playground>

### With Range

By using `value` as an array of `[number, number]`, it will automatically transform into a range slider. The minimum range is the value of the step

<Playground>
  <Slider
    value={[30, 60]}
    step={5}
    minimumValue={0}
    maximumValue={100}
    onValueChange={value => console.log('onValueChange', value)}
    onSlidingComplete={value => console.log('onSlidingComplete', value)}
    onSlidingStart={value => console.log('onSlidingStart', value)}
  />
</Playground>
