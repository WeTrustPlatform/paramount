# Badge

Display to indicate status or feat of service

### Usage

<Playground>
  <Box>
    <Badge title="Badge" color="blue" isSolid size="large" shape="pill" />
  </Box>
</Playground>

### Props

<Props of={Badge} />

### Customization

Using `getStyles` prop

```tsx
BadgeStylesProps {
  shape: ContainerShape;
  size: ControlSize;
  color: FillColor;
  isSolid: boolean;
}

BadgeStyles {
  textStyle: TextStyle;
  containerStyle: ViewStyle;
}

getStyles={(BadgeStylesProps, Theme) => BadgeStyles}
```

Markup

```tsx
<View containerStyle>
  <Text textStyle>{title}</Text>
</View>
```
