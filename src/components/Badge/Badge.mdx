# Badge

### Usage

<Playground>
  <Box>
    <Badge color="blue" isSolid size="large" shape="pill">

### Customization

Using `getStyles` prop

```
BadgeStylesProps {
  shape: ContainerShape;
  size: ControlSize;
  color: FillColor;
  isSolid: boolean;
}

BadgeStyles {
  textStyle: TextStyle;
  containerStyle: ViewStyle;
}

getStyles={(BadgeStylesProps, Theme) => BadgeStyles}
```
