# Heading

Composes React Native's `Text` component

### Usage

<Playground>
  <Box>
    <Heading size="xxxlarge">xxxlarge</Heading>
    <Heading size="xxlarge">xxlarge</Heading>
    <Heading size="xlarge">xlarge</Heading>
    <Heading size="large">large</Heading>
    <Heading size="medium">medium</Heading>
    <Heading size="small">small</Heading>
  </Box>
</Playground>

### Props

<Props of={Heading} />

### Customization

Using `getStyles` prop

```
HeadingStyles {
  headingStyle: TextStyle;
}

getStyles={(HeadingProps, Theme) => HeadingStyles}
```

Markup

```tsx
<Text headingStyle />
```
