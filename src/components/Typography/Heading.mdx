# Heading

Composes React Native's `Text` component (i.e. you can pass its props)

**Note!!!** In web environment sometimes you need to wrap `Heading` in a `<Box>` or provide a parent node with `display: flex` and `flex-direction: column` for centering to work properly. In React Native it is non issue because all nodes be default have these properties

### Usage

<Playground>
  <Box>
    <Heading
      accessibilityLevel={3} // 1-6
      align="center"
      size="xxlarge" // small - xxxlarge or number
      color="primary" // or custom
      weight="600"
      getStyles={(props, theme) => ({
        headingStyle: {},
      })}
    >
      Lorem ipsum dolar set amet
    </Heading>
  </Box>
</Playground>
