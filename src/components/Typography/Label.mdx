# Label

Composes `Text`

## Usage

<Playground>
  <Box>
    <Label>Lorem ipsum dolar set amet</Label>
  </Box>
</Playground>

## Props

<Props of={Label} />

## Customization

Using `getStyles` prop

```
LabelStyles {
  labelStyle: LabelStyle;
}

getStyles={(LabelProps, Theme) => LabelStyles}
```

Markup

```tsx
<Text labelStyle />
```
