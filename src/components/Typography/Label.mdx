# Label

Composes `Text`

### Usage

<Playground>
  <Box>
    <Label>Lorem ipsum dolar set amet</Label>
  </Box>
</Playground>

### Props

<Props of={Label} />

### Customization

Using `getStyles` prop

```
TextStyles {
  textStyle: TextStyle;
}

getStyles={(TextProps, Theme) => TextStyles}
```

Markup

```tsx
<Text textStyle />
```
