# Typography

All Typogrography components compose of React Native's `Text` component (i.e. you can pass its props)

`color` prop takes preset color names from the theme (which includes `primary`, `secondary`, `success` etc.), it also can take a custom color such as `#16a054`

## Heading

### Usage

<Playground>
  <Heading
    accessibilityLevel={3} // 1-6
    align="left"
    size="xxlarge"
    color="primary"
    getStyles={(props, theme) => ({
      headingStyle: {},
    })}

    Lorem ipsum dolar set amet
