# Typography

All Typogrography components compose of React Native's `Text` component (i.e. you can pass its props)

## Heading

### Usage

<Playground>
  <Heading
    accessibilityLevel={3} // 1-6
    align="left"
    size="xxlarge"
    getStyles={(props, theme) => ({
      headingStyle: {},
    })}
  >
    A red flair silhouetted.
  </Heading>

## Text

Also, all components below `Text` composes it, thus they will inherit this component's props

### Usage

<Playground>
  <Text
    color="primary"
    align="left"
    size="large" // small - large
    fontFamily="text"
    isBold
    isItalic
    isInline
    transform="uppercase"
    getStyles={(props, theme) => ({
      textStyle: {},
    })}
  >
    A red flair silhouetted.
  </Text>
</Playground>
