<Props of={Paragraph} />

## Customization

Using `getStyles` prop

```
ParagraphStyles {
  textStyle: ParagraphStyle;
}

getStyles={(ParagraphProps, Theme) => ParagraphStyles}
```

Markup

```tsx
<Paragraph paragraphStyle />
```
