# Positioner

A base component for manipulating the position of the element on the screen. Used by `Popover`

### Usage

<Playground>
  <Toggle>
    {({ on, toggle }) => (
      <Positioner
        isVisible={on}
        onClose={toggle}
        isDynamicContent={false}
        position="top-right"
        content={props => (
          <Box width={200} height={200}>
            <Text>
              Content that can be anything. By default it has no styling at all.
              You can use `Popover` that comes with predefined styles.
              Positioner is used as a base for `Popovers`, `Tooltips`,
              `DropdownMenus`
            </Text>
          </Box>
        )}
        isFullWidth={false}
        parentHeight={1000}
        // Use this prop to specify the position of a component u want to wrap with the positioner
        // targetMeasurements={{}}
        getStyles={(props, theme) => ({
          positionerStyle: {},
          modalContainerStyle: {},
        })}
      >
        <Button onPress={toggle} title="Usage" />
      </Positioner>
    )}
  </Toggle>
</Playground>
