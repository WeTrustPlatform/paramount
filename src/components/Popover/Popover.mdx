# Popover

Composes `Positioner`

### Usage

<Playground>
  <Box height={250}>
    <Toggle>
      {({ on, toggle }) => (
        <Popover
          isVisible={on}
          position="bottom"
          content={
            <Box width={200} height={200}>
              <Text>
                Due to variety and complexity involved in detecting touch
                outside of the component on the web and in native, managing its
                open and closed state is in your control
              </Text>
              <Button
                appearance="outline"
                color="danger"
                onPress={toggle}
                title="Close popover"
              />
            </Box>
          }
        >
          <Button onPress={toggle} title="Open popover" />
        </Popover>
      )}
    </Toggle>
  </Box>
</Playground>

### Props

<Props of={Popover} />

### Customization

Using `getStyles` prop

```tsx
PopoverStyles {
  popoverStyle: ViewStyle;
  positionerStyle: ViewStyle;
  containerStyle: ViewStyle;
}

getStyles={(PopoverProps, Theme) => PopoverStyles}
```

Markup

```tsx
<>
  <View positionerStyle>
    <View popoverStyle>{content}</View>
  </View>
  <View containerStyle>{children}</View>
</>
```
