<Props of={Divider} />

### Customization

Using `getStyles` prop

```
DividerStyles {
  dividerStyle: TextStyle;
}

getStyles={(DividerProps, Theme) => DividerStyles}
```

Markup

```tsx
<View dividerStyle />
```
