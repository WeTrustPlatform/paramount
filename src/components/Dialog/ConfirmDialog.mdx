# ConfirmDialog

Opposed to `Dialog` (below), `ConfirmDialog` has predefined styling and handlers; however, it still can pass props from `Dialog`

### Usage

<Playground>
  <Toggle initial={false}>
    {({ on, toggle }) => (
      <Box>
        <ConfirmDialog
          title="Title"
          isVisible={on}
          onRequestClose={toggle}
          cancelLabel="Cancel"
          confirmLabel="Confirm"
          header={undefined}
          footer={undefined}
          useHistory={false}
          onConfirm={() => console.log('Confirmed')}
          getStyles={(props, theme) => ({
            modalContainerStyle: {},
            containerStyle: {},
            bodyStyle: {},
          })}
        >
          <Text>Add custom content here</Text>
        </ConfirmDialog>
        <Button onPress={toggle} title="Open dialog" />
      </Box>
    )}
  </Toggle>
</Playground>
