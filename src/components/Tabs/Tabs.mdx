# Tabs

Passing `activeTabIndex` and `onPress` will alternate to controlled usage (manually set tab)

You can use a custom `Tab` component as long as it accepts `isActive`, `onPress` props

### Usage

<Playground>
  <Tabs
    defaultActiveTabIndex={1}
    activeTabIndex={undefined}
    onPress={undefined}
    shouldFit
    getStyles={(props, theme) => ({
      containerStyle: {},
    })}
  >
    {new Array(4).fill(0).map((v, i) => (
      <Tab key={i} title={`Tab ${i + 1}`} />
    ))}
  </Tabs>
</Playground>
