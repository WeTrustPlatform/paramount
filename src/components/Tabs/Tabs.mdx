# Tabs

You can use a custom component in place of `Tab` component, though keep in mind you need to manage `isActive` and `onPress` by yourself

## Usage

<Playground>
  <State initial={{ activeTabIndex: 0 }}>
    {({ state, setState }) => (
      <Tabs
        activeTab={state.activeTabIndex}
        onTabChange={tab => setState({ activeTabIndex: tab })}
        tabs={new Array(4).fill(0).map((v, i) => ({
          title: `Tab ${i + 1}`,
        }))}
      />
    )}
  </State>
</Playground>

## Tabs Props

<Props of={Tabs} />

## Tabs Customization

Using `getStyles` prop

```tsx
TabsStyles {
  containerStyle: ViewStyle;
}

getStyles={(TabsProps, Theme) => TabsStyles}
```

Markup

```tsx
<View style={containerStyle}>{tabs}</View>
```

## Tab Customization

Using `getStyles` prop

```tsx
TabStyles {
  containerStyle: ViewStyle;
  touchableStyle: ViewStyle;
  textStyle: TextStyle;
}

getStyles={(TabProps, Theme) => TabStyles}
```

Markup

```tsx
<View style={containerStyle}>
  <Button touchableStyle textStyle>
</View>
```
