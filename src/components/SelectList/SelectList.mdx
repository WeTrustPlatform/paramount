# SelectList

SelectList that supports single value or multiple values selected at the same item. `SelectList` is a compound component, which means that its children can be any component, and it will implicitly inject props such as `isSelected` or `onSelect` to them.
`SelectListItem` composes `ListItem` and `SelectList` composes `FlatList`

### Usage

<Playground>
  <SelectList
    isMulti={false} // Single select
    selectedValue="c"
    onValueChange={(newValue, itemIndex) => {
      console.log(newValue, itemIndex);
    }}
  >
    <SelectListItem
      label="Usage"
      value="c"
      // Props below are passed from `SelectList`, so it is not necessary to pass them here
      // isSelected
      // onSelect
      // index
    />
  </SelectList>
</Playground>
